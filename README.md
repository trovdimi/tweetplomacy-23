# Tweetplomacy 23 – An Annotated Collection of Tweets Outlining Strategies of Political Risk Communication during Global Crises (2018-2023)

Welcome to the repository for the analysis notebooks accompanying the paper:

**Tweetplomacy 23 – An Annotated Collection of Tweets Outlining Strategies of Political Risk Communication during Global Crises (2018-2023)**

## Overview

This repository contains code and Jupyter notebooks for analyzing the [Tweetplomacy 23](https://doi.org/10.7802/2985) dataset: an annotated collection of tweets by political actors during major global crises between 2018 and 2023. The analyses here reproduce and extend methods and results presented in the associated paper.

- **Dataset DOI:** [10.7802/2985](https://doi.org/10.7802/2985)
- **Webpage:** [data.gesis.org/tweetplomacy-23](https://data.gesis.org/tweetplomacy-23)
